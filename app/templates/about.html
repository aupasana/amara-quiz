About this site
This site is developed by Ajit Krishnan
The sources are on github
Data set: amara kosha mulam on github
Data set: amara kosha word csv file on github
Data set: babylon dictionary files on github

Features
  • Varga view with sanskrit summary, translations and pratipadikas for each line
  • Sloka view with detailed information of all the words therein
  • Pada view with information of the pada as well as all synonyms
  • Search the amara kosha with full wildcard support
  • Search the apte dictionary (including compound words) with full wildcard support
  • Quizzes (vocabulary drills) based on the amara kosha
  • View the amara kosha in devanagari, kannada, telugu or malayalam
  • English translations [in progress]
  • Telugu translations [in progress]
  • Link each sloka to the relevant audio files with correct offset [in progress]
