The functionality below is built on top of the UoHyd amara kosha dataset.

Enter search text in देवनागरी or itrans format. Examples: "indra*", "वि*वत्", "*dra", "शिव शक्ति"

Note: the sloka numbers in the JNU dataset will not match the main dataset exactly.

This application is a training aid for those who are memorizing the amarakosha. It supports simple quizzes as well as basic search of the text. The sources are on github.
